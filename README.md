## 摘要 | Abstract

基于携程旅行网 **3050 条游客评价数据**（2016–2026），对盐城中华麋鹿园（5A 景区）进行评分分布、时间趋势、季节性特征、用户画像与文本情感的多维度分析。采用"爬取 → 清洗 → 分析 → 可视化 → 报告"全链路架构，通过 **jieba** 中文分词挖掘好评/差评核心关键词，**matplotlib** 生成 11 张静态分析图表，最终以 **python-docx** 自动生成结构化 Word 分析报告。

> Based on 3050 tourist reviews (2016–2026) from Ctrip, this project conducts multi-dimensional analysis of Yancheng China Milu Park (5A scenic area) covering rating distribution, time trends, seasonality, user profiles and text sentiment. It adopts a full-pipeline architecture from scraping → cleaning → analysis → visualization → reporting, mining positive/negative keywords via jieba, generating 11 static charts via matplotlib, and finally producing structured Word reports through python-docx.

## 背景与动机

在景区运营管理中，游客在线评价是最直接、最真实的口碑数据源。然而大量景区仅关注评分均值的表面指标，忽略了评价文本中蕴含的 **体验痛点、核心卖点与客群偏好**。

本项目试图回答以下问题：

1. 景区总体口碑如何？评分在景色、趣味、性价比三个维度上是否存在结构性差异？
2. 评价量随时间如何变化？是否存在明显的季节性规律？旺季/淡季的时间窗口在哪里？
3. 核心客群来自哪些地区？高等级用户的评价倾向是否与普通用户不同？
4. 好评的核心驱动力是什么？差评的集中痛点在哪里？——这些洞察如何转化为运营改进建议？

## 关键发现总结

### 发现一：口碑整体优秀，但性价比是短板

景区总体评分 4.50/5.0，好评率 88.4%。但三个维度存在 **梯度落差**——景色 4.51 > 趣味 4.45 > **性价比 4.38**。性价比是三个维度中最低的，差评关键词中"门票""排队""小时"频繁出现，说明游客对价格感知和等待体验有明确不满。

### 发现二：2025 年评价量爆发式增长

2025 年评价量达 **1727 条**，是 2024 年(93 条)的 **18 倍**。这一增长与景区知名度提升、社交媒体传播效应、平台评价激励政策等多重因素相关。2026 年已累积 1006 条，热度仍在持续。

### 发现三：旺季窗口集中在夏季 4–6 月

夏季（4–6 月）评价量占全年 **40.3%**，是最核心的旺季窗口。这与麋鹿活跃期（春季产仔、夏季鹿群可见度高）高度吻合。秋季（7–9 月）反而是最淡的季节，仅占 15.2%——运营方应在秋季推出差异化活动以拉平客流波动。

### 发现四：客群高度集中于长三角

江苏(51.5%) + 上海(15.5%) + 浙江(6.1%) 合计占 **72.6%**，客群高度集中于长三角邻近省份。景区当前依赖本地与近程客源，远程客源（广东 1.4%、北京 1.9%）占比极低——这意味着 **远程市场有较大增长潜力**。

### 发现五：近距离互动与亲子体验是核心竞争力

好评关键词中"观光车""胡萝卜""近距离""孩子"构成核心体验词簇。景区的差异化优势不是单纯的自然观光，而是 **可参与的近距离互动体验**——这一卖点应在营销中强化突出。

### 发现六：排队是最大运营痛点

差评关键词中"排队"(39 次)和"小时"(36 次)高频出现，远超其他差评词。高峰期排队等候时间长是游客体验的主要破坏因素，应通过预约分流、增加观光车班次、错峰定价等方式缓解。

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/Liuwenke-hub/ctrip-milu-park-analysis.git
cd ctrip-milu-park-analysis

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行全部脚本（按顺序）

# ① 采集评价数据（可选 — data/ 已包含 3050 条数据）
python src/scrape_ctrip_reviews.py
#    -> data/ 中华麋鹿园_评价数据.csv / .xlsx / _raw.json

# ② 生成静态分析图
python src/generate_charts.py
#    -> charts/ 01~11_*.png（11 张 300 DPI 图表）

# ③ 生成 Word 正式分析报告
python src/generate_report.py
#    -> output/ 中华麋鹿园_评价数据分析报告.docx
#    （含 11 张图表 + 6 章文字分析 + 改进建议）

# ④ 生成 HTML 在线报告
python src/generate_html_report.py
#    -> output/ 中华麋鹿园_评价数据分析报告.html
#    浏览器打开即可查看，宽屏自适应布局

# ⑤ 生成 pyecharts 交互式仪表盘（可选）
python src/analyze_reviews.py
#    -> output/ 中华麋鹿园_评价数据分析仪表盘.html

# ⑥ 交互式 Notebook（可选：pip install jupyter 后启动）
#    jupyter notebook notebooks/中华麋鹿园_评价数据分析.ipynb
```

## 摘要 | Abstract

基于携程旅行网 **3050 条游客评价数据**（2016–2026），对盐城中华麋鹿园（5A 景区）进行评分分布、时间趋势、季节性特征、用户画像与文本情感的多维度分析。采用"爬取 → 清洗 → 分析 → 可视化 → 报告"全链路架构，通过 **jieba** 中文分词挖掘好评/差评核心关键词、**SnowNLP** 情感极性模型补充文本语义视角，**matplotlib** 生成 12 张静态分析图表，最终以 **python-docx** 自动生成结构化 Word 分析报告。

> Based on 3050 tourist reviews (2016–2026) from Ctrip, this project conducts multi-dimensional analysis of Yancheng China Milu Park (5A scenic area) covering rating distribution, time trends, seasonality, user profiles and text sentiment. It adopts a full-pipeline architecture from scraping → cleaning → analysis → visualization → reporting, mining positive/negative keywords via jieba, classifying sentiment polarity via SnowNLP, generating 12 static charts via matplotlib, and finally producing structured Word reports through python-docx.

## 背景与动机

在景区运营管理中，游客在线评价是最直接、最真实的口碑数据源。然而大量景区仅关注评分均值的表面指标，忽略了评价文本中蕴含的 **体验痛点、核心卖点与客群偏好**。

本项目试图回答以下问题：

1. 景区总体口碑如何？评分在景色、趣味、性价比三个维度上是否存在结构性差异？
2. 评价量随时间如何变化？是否存在明显的季节性规律？旺季/淡季的时间窗口在哪里？
3. 核心客群来自哪些地区？高等级用户的评价倾向是否与普通用户不同？
4. 好评的核心驱动力是什么？差评的集中痛点在哪里？——这些洞察如何转化为运营改进建议？

### 分析口径

- **评分分布 / 平均评分**：基于全部 3050 条（总评分无缺失）。
- **分项维度均值（景色 / 趣味 / 性价比）**：基于约 91–92% 有值样本，缺失样本按 pandas 默认忽略（NaN）；因此维度均值与"全量均值"存在约 0.01 的口径差异属正常。
- **用户等级分布**：饼图基于有值的 2547 条（占全量 83.5%；16.5% 缺失未计入），扇区百分比之和恰为 100%（分母已修正为有效样本，避免"占全量 3050"导致的语义偏差）。
- **文本关键词**：采用 jieba 分词 + 停用词过滤；核心主题词「麋鹿」已**保留**（单景区分析中最具代表性，应在词云/高频词中正常呈现）。高频词与词云基于全量评论；好评/差评关键词分别基于评分 ≥4 星（2698 条）与 ≤2 星（157 条）的子集。
- **情感分析方法（双视角）**：
  - *评分分段视角*：将总评分 ≥4 星（2698 条）与 ≤2 星（157 条）分别做关键词词频对比，属于关键词共现/频次分析，用于定位"高分好评 vs 低分差评"的体验差异；差评样本仅 157 条（占 5.1%），样本量较小，相关结论用于提示方向而非统计推断。
  - *文本极性视角（新增）*：引入 **SnowNLP** 中文情感极性模型，对每条评论做 0-1 语义打分（正面≥0.6 / 负面≤0.4 / 中性之间）。模型判定负面 17.3%（528 条），显著高于按星级的差评率（5.1%），并识别出 **331 条"高分但模型判负面"的隐性不满评价**——说明不少高分游客的文字中仍夹带排队、价格等负面体验，需在好评中进一步挖掘改进线索。SnowNLP 基于电商语料训练，对旅游场景为近似估计，结论作为方向性参考。

**2025 年为何爆发？外部归因（检索时间：2026-07-21）**

本数据集只呈现"增长了多少"（2025 年 1727 条 ≈ 2024 年 80 条的 21.6 倍），并不直接解释"为什么"。结合公开报道，增长主要由以下结构性因素驱动：

- **运营体制改革（2024.4 起）**：盐城市属国企燕舞集团接手中华麋鹿园市场化运营，游客数同比增长近 40%，并提前完成年度营收目标。〔来源：《新华日报》2026-01-15《燕舞集团用"新"破题激活文旅新动能》；江苏省文旅厅 2025 年度 A 级景区复核通报〕
- **超级 IP 引爆（2024.6）**：第十二届"鹿王争霸"大型融媒体直播，26 小时不间断、全网观看量破 2000 万、触达近亿，成功引爆话题。〔来源：同上《新华日报》〕
- **"苏超"体育联赛破圈（2025）**：赛事带动城市标志性景区，盐城中华麋鹿园抖音团购订单消费环比 +353%、金额 +368%。〔来源：腾讯新闻《美猴王、鹿王、南通狼等"苏超"IP破圈，相关景区抖音团购订单消费暴涨》（2025）〕
- **文旅+科技（2025 国庆）**：VR 项目"麋鹿的千年传奇"上线，游客平均停留 1.5h→3h，二次消费占比 +25 个百分点。〔来源：同上《新华日报》；今日头条景区 5A 复核报道〕
- **新媒体矩阵 + 亲子研学**：抖音 / 小红书 / 公众号持续传播"麋鹿"IP；2025 五一"邂逅萌趣麋鹿"系列活动拉动亲子客流。〔来源：中国江苏网 2025-05-04《燕舞文旅各景区"花式玩法"助推盐城假日旅游持续升温》〕
- **5A 复核通过（2025）**：景区通过国家 5A 级旅游景区评定性复核，品牌公信力提升。〔来源：江苏省文旅厅通报 / 今日头条〕

**差评核心关键词（评分≤2星，157 条）**：麋鹿(72)、观光车(40)、排队(39)、看到(25)、体验(25)、门票(23)、工作人员(21)、电瓶车(21)、景点(18)、游客(18)

> → 在剔除"麋鹿/观光车"等通用高频词后，差评的**集中痛点**指向：**排队等候时间过长(39) + 门票/性价比偏低(门票23) + 服务态度不佳(工作人员21) + 电瓶车/观光车接驳体验(电瓶车21)**

**文本极性视角（SnowNLP 模型）**：对全部 3050 条评论做情感极性打分，分布为 **正面 75.2%（2295）、中性 7.4%（227）、负面 17.3%（528）**。与"按星级判差评仅 5.1%"相比，文本语义层面的负面比例高出逾 3 倍——典型如"景色很好，但排队两小时""鹿很可爱，就是门票不值"，属于高分低极性。其中 **331 条 4-5 星评价被模型判为负面（隐性不满）**，是运营方最易忽视的改进金矿。

## 关键发现总结

### 发现一：口碑整体优秀，但性价比是短板

景区总体评分 4.50/5.0，好评率 88.5%。但三个维度存在 **梯度落差**——景色 4.52 > 趣味 4.45 > **性价比 4.39**。性价比是三个维度中最低的，差评关键词中"门票""排队"频繁出现，说明游客对价格感知和等待体验有明确不满。

### 发现二：2025 年评价量爆发式增长

2025 年评价量达 **1727 条**，是 2024 年(80 条)的 **约 21.6 倍**。这一增长与景区知名度提升、社交媒体传播效应、平台评价激励政策等多重因素相关。2026 年已累积 1019 条（截至 2026-07），热度仍在持续。

### 发现三：旺季窗口集中在夏季 4–6 月

夏季（4–6 月）评价量占全年 **40.3%**，是最核心的旺季窗口。这与麋鹿活跃期（春季产仔、夏季鹿群可见度高）高度吻合。秋季（7–9 月）反而是最淡的季节，仅占 15.7%——运营方应在秋季推出差异化活动以拉平客流波动。

### 发现四：客群高度集中于长三角

江苏(51.4%) + 上海(15.7%) + 浙江(6.0%) 合计占 **73.1%**，客群高度集中于长三角邻近省份。景区当前依赖本地与近程客源，远程客源（广东 1.4%、北京 1.9%）占比极低——这意味着 **远程市场有较大增长潜力**。

### 发现五：近距离互动与亲子体验是核心竞争力

好评关键词中"观光车""胡萝卜""近距离""孩子""接触""小朋友"构成核心体验词簇。景区的差异化优势不是单纯的自然观光，而是 **可参与的近距离互动体验**——这一卖点应在营销中强化突出。

### 发现六：排队是最大运营痛点

差评关键词中"排队"(39 次)居首，其次为"门票"(23)、"工作人员"(21)、"电瓶车"(21)。高峰期排队等候时间长是游客体验的主要破坏因素，应通过预约分流、增加观光车/电瓶车班次、错峰定价等方式缓解；"门票/性价比"与"服务态度"亦需同步优化。

### 发现七：文本情感比星级更严苛，隐性不满值得深挖

SnowNLP 模型判定负面评论占 **17.3%**（528 条），远高于按星级的差评率 **5.1%**——大量"高分低极性"评价（如"景色不错，但排队太久""鹿可爱，门票偏贵"）在星级上被掩盖。其中 **331 条 4-5 星评价被模型判为负面（隐性不满）**，是运营方最易忽视的改进金矿：建议在好评中叠加语义监控，对"高分但提及排队/价格/服务"的评价定向回访，将隐性不满转化为复购与口碑。

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/Liuwenke-hub/ctrip-milu-park-analysis.git
cd ctrip-milu-park-analysis

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行全部脚本（按顺序）

# ① 采集评价数据（可选 — data/ 已包含 3050 条数据）
python src/scrape_ctrip_reviews.py
#    -> data/ 中华麋鹿园_评价数据.csv / .xlsx / _raw.json

# ② 生成静态分析图
python src/generate_charts.py
#    -> charts/ 01~12_*.png（12 张 300 DPI 图表）

# ③ 生成 Word 正式分析报告
python src/generate_report.py
#    -> output/ 中华麋鹿园_评价数据分析报告.docx
#    （含 12 张图表 + 6 章文字分析 + 改进建议）

# ④ 生成 HTML 在线报告
python src/generate_html_report.py
#    -> output/ 中华麋鹿园_评价数据分析报告.html
#    浏览器打开即可查看，宽屏自适应布局

# ⑤ 生成 pyecharts 交互式仪表盘（可选）
python src/analyze_reviews.py
#    -> output/ 中华麋鹿园_评价数据分析仪表盘.html

# ⑥ 交互式 Notebook（可选：pip install jupyter 后启动）
#    jupyter notebook notebooks/中华麋鹿园_评价数据分析.ipynb
```

> 注意：`output/` 目录下的报告与仪表盘为运行脚本后的生成物，已通过 `.gitignore` 忽略、**不随仓库提交**，克隆后请运行 ③④⑤ 在本地生成。
